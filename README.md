# Up Arrow Inc Corporate Website

A clean, modern corporate website with professional design and responsive layout.

## What's Included

```
├── index.html          # Homepage with hero, services, testimonials
├── services.html       # Detailed services page
├── about.html         # Company information and team
├── contact.html       # Contact form and information
├── assets/
│   ├── css/
│   │   └── styles.css # Main stylesheet with responsive design
│   ├── js/
│   │   └── main.js    # Interactive features and form handling
│   └── images/
│       └── *.svg      # Service icons and testimonial images
```

## Features

- **Modern Design**: Professional styling with smooth animations
- **Responsive Layout**: Perfect on mobile, tablet, and desktop
- **SEO Optimized**: Proper meta tags and semantic HTML
- **Interactive**: Contact forms with validation
- **Fast Loading**: Pure HTML/CSS/JS with no dependencies

## Easy ColdFusion Conversion

To convert to ColdFusion:
1. Rename `.html` files to `.cfm`
2. Add ColdFusion tags where needed (`<cfoutput>`, `<cfif>`, etc.)
3. Keep the existing CSS and JavaScript
